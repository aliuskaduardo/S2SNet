Sequence to Star Network 4.0

S2SNet is a Python/wxPython application with Graphviz as plotting back-end. S2SNet helps you to transform the character sequences/strings in Star to Network topological indices and visualize the resulted graphs. These numbers can be used for a statistical analysis of your initial data. The initial use of this program is the creation of statistical models for protein sequences.

What S2SNet can do

  • Transform (one-) character sequences in topological Star Networks indices;
  • Transform number data in sequences (No2Seq);
  • Transform N-character sequence in 1-character sequence (change the code);
  • Edit/View your input and output TXT files;
  • Create DOT language files;
  • Plot and Display Networks PNG images.

How to use S2SNet

In the main windows you can choose the calculation parameters and the input/output files:
  • Parameters: embedded network, the use of the weight for each character; Markov normalization of the connectivity matrixes; if you want details of the calculations containing all the intermediate matrices and other info; power of the connectivity matrices and of some indices (max. 5); networks plotting support,
  • Input files: sequences, groups and weights files (the columns should be TAB-delimited, not containing "\" or "/"),
  • Output files: results and details files,
  • Display mode for Network plots: sequence to display and type of drawing application (for embedded case use circo for better results).

All the options have default values in order to perform minimum of calculations.

The processing of the sequences can be seen in a console window. If you will close it, all the application windows will close too.

The buttons can be found in menu too (excluding Display). In addition, the menu contains the call for Notepad text editor that you need to view/edit your input/output files or create new ones.

Each group has a different colour. If you need to obtain different plots, you can find the DOT files (one for each sequence) and the Graphviz executables (dot, circo, twopi, neato, fdp) in the "dot" folder.

The Calculations menu allows you to transform your data in the S2SNet format (1-character string):

Numbers to Sequence - Transforms your numbers in a sequence (the numbers must be TAB separated); you can choose:
  • Parameters: minimum and maximum values of your data, number of groups you need (maximum 80); you have a GET button if you want to use the mimimum and maximum calculated from your entire data,
  • Input files: number (data) file, 
  • Output files: sequence files, group file and interval file (description of the group range).
Note: this can be used to transform the protein mass spectra numbers in sequences in order to calculate Star Networks indices.

N to 1-Character Sequence - Transform your N-character sequences in 1-character sequences; you can set the following:
  • Input files: N-character file (initial file), code file (the equivalence between N-character and 1-character; ex: ALA=A)
  • Output files: 1-character files for S2SNet (final file) and group file (one item groups).
Note: this calculation can be used to transform sequences that contain items described by 3-letter codes of the amino acids into sequences of 1-letter code; another example is the analyse of the DNA sequences with the Star Networks (the 3-letter codes are transformed in 1-letter codes gaved by S2SNet).

If S2SNet will raise any error, you can see it in the same console and please send us in order to fix it.  
 

Star to network indices

Your data will be used to calculate the following indices:

- Shannon Entropy of the n Markov Matrices (Sh)


where pi are the ni elements of the vector p resulted from the matrix multiplication of the powered Markov normalized matrix ( ni x ni ) and a vector ( ni x 1) with each element equal with 1/ni;

- The traces of the n connectivity matrices (Tr)
 
where n = 0 … power, M = connectivity matrix (i*i dimension); ii = ith diagonal element;

- Harary number (H)

where dij = elements of the distance matrix, mij = elements of the M connectivity matrix, wj = weight elements, nw = for selection (1) or no selection (0) of weights calculations;

- Wiener index (W)
  
- Gutman topological index (S6)
  
where degi = elements of the degree matrix;

- Schultz topological index (non-trivial part) (S)

- Moreau-Broto, Autocorrelation of Topological Structure (ATSn, n=1…power); only with weights included

where dpnij = elements of the pair distance matrix when the distance is n;

- Balaban distance connectivity index (J)

where nodes/edges = node/edge numbers in the Star Network;

- Kier-Hall connectivity indices
 

 
 

- Randic connectivity index
